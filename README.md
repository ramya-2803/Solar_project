# Solar Panel Optimization System

## Description
This project predicts solar irradiance and finds optimal panel tilt angle using ML.

## Features
- Optimal tilt calculation
- Power vs tilt graph
- 3D panel visualization
- PDF export

## How to Run

1. Go to backend folder:
   cd backend

2. Install dependencies:
   pip install -r requirements.txt

3. Run server:
   python app.py

4. Open browser:
   http://127.0.0.1:5000/

## Tech Stack
- Flask
- Machine Learning (XGBoost)
- Pandas, NumPy
- Matplotlib
